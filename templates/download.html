Downloading

Downloading darm is as simple as cloning the Git repository at Github using the following command: 

git clone git://github.com/jbremer/darm.git

Of course you can also download the zip file of the master branch, the latest stable release, directly.

Compiling

Compiling darm is as easy as running the make command inside the newly created darm directory.
Darm has been tested to compile correctly on Windows, Linux, and Mac OS X. (For windows, please install mingw32.) If, for some reason, darm does not compile correctly for you, do not hesitate to contact us in one way or another!
